Todoチャレンジ

概要

  • 従来のTodo listにSNS機能とポイント制のチャレンジシステムを組み合わせたアプリ
  • ユーザーが自分のタスク達成に対して「賭け」をすることでモチベーション向上を図る

仕組み

  • 各ユーザーには初期ポイントが付与される
  • 毎日、その日のタスク（1〜複数の目標）を設定できる
  • タスクにポイントを賭ける（賭けるポイント数はユーザーが自由に決定）
  • 日の終わりに自己評価を行い、達成度を記録する
    • （例：目標が2つ設定された場合，目標Aは8割できた，Bは4割できた　→　達成度60%）

機能

ログイン

  • IDとパスワードでログインする

一覧を見る

詳細結果を見る

決算

  • 達成度に応じてポイント還元率が変動
    • 例：50%達成 → 同額返還，70%達成 → 1.2倍返還，100%達成 → 2倍返還

SNS

  • 日々の結果ページにコメント欄を設置
  • ユーザー同士で応援やアドバイスのコメントが可能
  • 他ユーザーへのコメント投稿でポイント獲得（日単位で上限あり）
